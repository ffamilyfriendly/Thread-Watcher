# internationalization

Thread-Watcher is in servers across many different countries and languages. While it would be simplest for everyone to speak Swedish, it did not quite work out that way.

## Directory structure

Bot locale files are located in [/bot/locales](/bot/locales/) and are named after their country code (ex: en for english or sv for swedish). Each directory has a `common.json` file which defines all the translated language strings and also a `md` folder that allows an easier way to manage long strings.

## Markdown files

Editing long strings directly in a JSON file gets annoying quick. You can therefore extract extra dastardly strings into a markdown file in the `/[language]/md` folder!
The i18n build script will automatically convert these files into text and insert them into the `common.json` file.

### File naming

Each file must be named as the i18n key it is defining. For example, if you want to define the language string `advanced.regex_helper`, you will need to call the file `advanced_regex_helper.md`
