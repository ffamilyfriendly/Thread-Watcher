The bot

Thread-Watcher was created just a few weeks after discord announced threads. The issue with the current state of threads is that they are automatically archived after 3 days without server boosts, until now. Thread watcher is a bot whose sole purpose is to keep threads un-archived for you without having to dish out any money for server boosts. The bot does not come packaged with any moderation tools, fun commands, meme commands, you name it. This bot keeps threads un-archived. That's it.
