The Bot

Thread-Watcher was created just a few weeks after discord announced threads. The issue with the current state of threads is that they are automatically archived after 3 days without server boosts, until now. Thread watcher is a bot whose sole purpose is to keep threads un-archived for you without having to dish out any money for server boosts. The bot does not come packaged with any moderation tools, fun commands, meme commands, you name it. This bot keeps threads un-archived. That's it.

The Commands

if this list is confusing, join the support server. I will gladly try to explain it better
  • /watch [thread] toggles auto un-archive of thread.
    • argument [thread] the thread to add/remove from the list
  • /threads lists all thread in your server that the bot is watching
  • /batch [action] [parent] [optional:pattern] watch/un-watch a lot of threads at the same time!
    • argument [action] either watch or un-watch
    • argument [parent] the channel whose threads you want to add/remove (categories accepted)
    • optional argument [pattern] thread name must match this pattern to have action applied. Wildcards (*) allowed.
      Exclamation mark (!) at beginning of string will reverse the pattern thus only adding threads whose name does not match the pattern

      Example

      /batch action:watch parent:#channel pattern:*important* will add all threads under #channel with name that includes important

      Example with exclamation mark

      /batch action:watch parent:#channel pattern:!*important* will add all threads under #channel with name that does not include important
